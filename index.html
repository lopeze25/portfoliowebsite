This website showcases my engineering 2213 design projects, including digital logic systems and my final design work.

Portfolio 1: 3-Bit Combinational Divider

3-Bit Combinational Divider

This project involved designing a 3-bit combinational divider using digital logic techniques. The system was built to perform division operations through combinational logic without relying on repeated subtraction with memory or registers.

Portfolio 2: 4-Bit Two's Complement Adder/Subtractor

4-Bit Two's Complement Adder/Subtractor

This project focused on the design of a 4-bit circuit capable of both addition and subtraction. The design used combinational logic to switch between operations and detect overflow.

Final Design Project: Morse Code Decoder

Morse Code Decoder

This project involved designing a system capable of decoding Morse code input into readable output. The decoder interprets timing-based input patterns and converts sequences of dots and dashes into corresponding alphanumeric characters.
